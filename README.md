# Create Sidebar

Projeto desenvolvido para praticar a criação de sidebars modernas utilizando HTML, CSS e JavaScript.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em desenvolvimento front-end, organização de layouts e construção de componentes de navegação lateral.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Sidebar interativa
- Navegação lateral
- Interface moderna
- Layout organizado
- Estrutura responsiva

## 📚 Conceitos praticados

- Flexbox
- Estruturação de layouts
- Responsividade
- Manipulação de interface
- Organização visual
